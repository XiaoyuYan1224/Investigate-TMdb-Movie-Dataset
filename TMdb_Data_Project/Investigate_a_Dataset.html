# For each genre in array g, find the popularity and release year of the movies in that genre. 
# Calculate the mean popularity of a single genre each year by using grouby function. 
# Plot a line plot of the popularity of each genre over time. X-axis:Year, Y-axis: Popularity)
# Since matplotlib has only 10 colors, two types of representation are used, which are lines and dashed lines. 
# Identify which genre has the largest change in popularity these years. 
genre = df['genres']

most_diff_min_max = 0
most_genre = ""

plt.figure(figsize=(18, 16), dpi= 80, facecolor='w', edgecolor='k')
for i in range (len(g)):
    A = df[genre.str.contains(g[i])]
    a = A[['popularity','release_year']]
    p = a.groupby('release_year').mean()
    p_mean = p['popularity'].tolist()
    year = A['release_year'].unique()
    year = np.sort(year)
    if i < 10:
        plt.plot(year, p_mean,'-o')
    else: 
        plt.plot(year, p_mean, '--')
    max = p.max()
    min = p.min()
    diff = float(max - min)
    if diff > most_diff_min_max:
        most_diff_min_max = diff
        most_genre = g[i]
plt.legend(g)
plt.title('The Popolarity of Each Genre Over Time')
plt.xlabel('Year')
plt.ylabel('Popularity')
plt.show()

Conclusions¶

From question 1, the trend of popularity change of each movie genre till 2015 is obtained. The genre that has the greatest fluctuation in popularity these years is the 'Western'. As the line chart shown, the popularity of 'Adventure','Western', 'Science Fiction' and 'Action' movie is having a increasing trend.

From question 2, the difference in revenue and budget of a single genre is shown in the bar chart above. 'Adventure' movies have the greatest revenue-budget difference.

Combining the two questions together, since the 'Adventure' movies have an increasing popularity and large revenue-budget difference, directors may be suggested to increase the production of adventure movies in the future.
